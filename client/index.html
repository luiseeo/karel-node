Karel

Learn programming now

Karel is a programming language designed to teach the basics of programming. You will be programming Karel, a virtual robot, and help him accomplish some tasks. Karel runs completely on the browser, you don't need to install or setup anything.

Quick overview

Karel's world

Karel lives in a grid, and he can move to any 2 adjacent cells on it. However, there could be walls blocking the passage between 2 cellls.

Karel can also place or pick beepers in these cells. Beepers can be collected by Karel in his beeper bag

Karel has to be careful to not crash!

Imperative programming

Karel can perform some actions to interact within his environment

  • Move forward
  • Turn left
  • Put beeper
  • Pick left
  • Turn off
pickbeeper(); move(); turnleft(); move(); putbeeper(); turnoff();

Conditions

We can teach Karel to perform some actions only when some conditions are met. For example, if the a wall is on front of him, he probably shouldn't move towards it :). Karel has some sensors to analyze his world

  • Compass. Karel can know if he is facing North, South, East or West.
  • Collision detector. Karel can know if a wall is front of him or in one of his sides.
// Move if front is clear if (frontIsClear) { move(); }

Loops

We can teach Karel to repeat some actions several times or to repeat them while a condition is met

  • Repeat a number of times
  • Repeat while a condition is met
// Face north while (!facingNorth) { turnleft(); }

Functions

If you are planning the same group of instructions again and again, write a function.

void turnright() { turnleft(); turnleft(); turnleft(); }

Recursion

When a function calls itself it is called recursion. Karel can learn advanced moves using recursion.

// Measures the distance to the next wall void measureDistance() { if (frontIsClear) { move(); measureDistance(); putbeeper(); } }
